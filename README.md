# HybrIQ Monday Starter

AI agents for Monday.com powered by [HybrIQ](https://hybriq.dev).

- **Board Automation** — AI-powered item change analysis and action suggestions
- **Item Summarizer** — Summarize item updates, comments, and status changes
- **Status Reporter** — Generate status reports from board data

## Quick Start

```bash
git clone https://github.com/developerlabsai/hybriq-monday-starter.git
cd hybriq-monday-starter && cp .env.example .env
npm install && npm run dev
```

Configure Monday.com webhooks to point to `POST /webhooks/monday`.
